Kudos!
Don't overreact, but we're now in React. After a lot of time in JQuery, we made yet another shift to learn React. Using this far more efficient system with it's virtual DOM, we recreated various functionalities from previous projects (custom ajax calls now made through Axios, dynamic generation, Bootstrap, and more) in a new setup for quicker renders. And just because I can, I also created an admin function to change the UI according to the login state that grants additional functionality to registered users.
Bamazon
Why stick with one database? Let's build another! Here, we switched from SQL to the far more flexible MongoDB. Interfacted with Mongoose to add some clear and controlling constraints, this app's basic functionality is to pull information from the database and render it to the page. I additionally added dynamically generated notifactions and interfaces so that the user could have a shopping experience similar to the real thing, all organized into a UI based on the Marvel comic theme. After all, when you get the word 'BAM!', what else can you do?
Employee Finder
Here, we move into the servers. No longer limited to local hosts and isolated environments, we incorporated Express server construction through Node to access a SQL database. Then using Ajax functionality, we built custom API calls to render data to and from the database to take in a personality test (built with the finest of JQuery, of course) and render the perfect office mate for the user. Give it a shot. It's good fun.
Eat Your Feelz
Group Projects. Fun times. This project combined two APIs in order to (1.) capture the user's picture with a built-in photo app made with HTML canvas, (3.) analyze the emotions of that picture, and (3.) recommend recipes based on those emotions. Multi-facted and beyond the scope of any individual, my focus was on the JavaScript, calling in the emotion information, building custom algorithms based on psychological and physiological research, then converting them into culinary terms usable by the next API. Give it a whirl for some delicious inspiration.
IEX Stock Trader
Ah, stocks, the cornerstone of our investment-based economy. Now well-versed in JavaScript and JQuery, we began using Ajax calls to reference external API's to gather a world of information. After all, what's the point in curating a portfolio if there's no portfolio to curate?
